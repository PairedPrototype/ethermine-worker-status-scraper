# ethermine-worker-status-scraper

## How to use

1. Get [NodeJS](https://nodejs.org/en/download/)
1. Clone this repo
1. cd into the cloned repo
1. run `npm install`
1. run `npm build`
1. run `node dist/index.js get "https://ethermine.org/miners/useridhere/dashboard"`
